# Kryptadium-Mod

The Kryptadium Mod adds for the moment only one ore, a new ore called kryptonite, it adds lots of items and some food you can get.

 

Don't forget to check the wiki if you need help!

 

# Items:

 

-Kryptonite Ore

-Kryptonite

-Kryptonite Sword

-Kryptonite Axe

-Kryptonite Apple

-And Much More
